Getting started

Structure your content in folders. The rule is - one folder contains one web page.
You can name the folders with space and national characters. eg: "Mötley Crüe"

Edit "index_vars.js" file

The file "index_vars.js" in each folder contains variables that are read by "index.htm to load page content. Open "index_vars.js" in each folder and modify the variable values for your page name.

VARS.name = "Page title"; (Page name, also used as H1 tag in "index.htm")
VARS.data = "page-readme"; (Data ID, that can be used as page identifier)
VARS.category = "Help"; (Category, for any purpose)
VARS.file_name = "readme.htm"; (file name with html text that is read by index.htm)

You can edit file "pages.htm" for pages/sub folder to show and made viewable.
