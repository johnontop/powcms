Optional downloads

- Optional PHP version 5.6.4 NTS VC11 Win32 for Windows XP/2003 and newer. vcredist VC11 for Windows XP service pack 3 included.
- pow-mini.zip, example of minimal version of POWCMS. This is created with CreateMiniDist.bat file in POWCMS root.
- customize.zip, example of needed files to customize POWCMS.
- pow-osx.zip, contains a bash script that might launch a web server and php on an Apple computer. But I know nothing about OSX.
