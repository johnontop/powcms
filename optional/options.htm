Optional files for downloads

Optional files are zip files that shall be extracted as folders in the root.

test.zip, contains test and demo pages
- test.zip, ccontains some test and demo pages to show the option module functionality.
- Just delete the folder test and do a System Update to show the correct menu.

Below files should be downloaded manually.

PHP version 5.6.4 NTS VC11 Win32 for Windows XP/2003
- Optional PHP version 5.6.4 NTS VC11 Win32 for Windows XP/2003 and newer. vcredist VC11 for Windows XP service pack 3 included.

pow-mini.zip, example of minimal version of POWCMS
- pow-mini.zip, example of minimal version of POWCMS. This is created with CreateMiniDist.bat file in POWCMS root.

customize.zip, example of needed files to customize POWCMS.
- customize.zip, example of needed files to customize POWCMS.

pow-osx.zip, contains a bash script
- pow-osx.zip, contains a bash script that might launch a web server and php on an Apple computer. But I know nothing about OSX.
